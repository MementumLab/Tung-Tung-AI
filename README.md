# Tung Tung Tung Sahur AI

A lightweight static mockup for Tung Tung Tung Sahur.

## Structure

- `index.html` - page shell and layout
- `src/styles/main.css` - visual design and responsive layout
- `src/scripts/app.js` - message rendering and interactions
- `src/data/conversations.js` - mock conversations and assistant text

## Preview

Open `index.html` directly in a browser.

## Notes

- No dependencies are required.
- The project is intentionally small, structured, and self-contained.
